# Stock & Ventas - Sistema de Gestión

Sistema de inventario y ventas para local comercial.

## Features

✅ Gestión de productos (CRUD completo)
✅ Control de stock con alertas
✅ Sistema de ventas con carrito
✅ Cálculo de vuelto automático
✅ Dashboard con estadísticas
✅ Responsive (mobile + desktop)

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase (PostgreSQL)
- Vercel

## Setup
```bash
npm install
# Configurar .env.local con Supabase keys
npm run dev
```
